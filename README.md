# 🧑‍🎓 Student Structure Lab (Управление данными студентов) Вариант - 18
Курсовая работа на основе ->
КОПИЯ - Лабораторная работа по работе со структурами в C++. Программа позволяет вести учёт студентов, их оценок и личных данных с сохранением в файл.

## 📋 Функционал

- ➕ Создание новой записи о студенте (ФИО, пол, группа, номер в списке, оценки)
- ✏️ Внесение изменений в существующую запись
- 📄 Вывод всех данных о студентах
- 🔍 Фильтрация по номеру группы
- 🏆 Топ студентов по среднему баллу
- ♀️♂️ Статистика по полу
- 🎓 Фильтрация по успеваемости (стипендия / хорошисты / отличники)
- 🔢 Поиск по номеру в списке

Данные автоматически сохраняются в текстовый файл.
